CloudShare


What is it?

CloudShare is a secure file storage and sharing service. All uploaded files are symmetrically encrypted with AES, decryptable only by the user. It was built on Ruby on Rails, a web-application framework built using the Ruby language.

Who built it?

CloudShare is a project created by myself and Frank Yu. The github repo can be found here.

The sign up page. The site would allow user's to create accounts, with an emphasis on securityâ€“passwords were hashed and salted with the bcrypt gem.

The front end portion of this site was built using twitter's bootstrap framework. Heroku was used to host the site.

Users all have a public and private key upon signup. The private key would be symmetrically encrypted using the user's own password, so that even us, the site devs, could have no access to it.

Files could be uploaded to the site, where they would be symmetrically encrypted using the user's public key. If the user wanted to download it, they would have to enter their password.

User's were allocated with a certain amount of storage available for them to use. Any more, and the site would stop the file from being saved upon the site.

CloudShare also supports file sharing. This is done in an encrypted manner through RSA.
